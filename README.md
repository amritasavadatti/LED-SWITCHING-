LED-SWITCHING-
==============

The system has one input switch (SW2 (connected to PJ1)) and two output LEDs (LEDs D1 (connected to PN1) and D2 (connected to PN2)). Overall functionality of this system is described in the following rules. 1) The system starts with the LED D1 ON and the LED D2 OFF. 2) If SW2 is pressed, then D1 will be turned off, and D2 will be toggled about every half second 3) If SW2 is released, then D2 will be turned off, and D1 will be toggled about every half second
